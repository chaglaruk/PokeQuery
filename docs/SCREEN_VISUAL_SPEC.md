# Screen Visual Spec

## Onboarding Screen
- **Background**: Dark night-map full background using `bg_night_map.webp`.
- **Title**: Large title area near the top.
- **Subtitle**: Placed directly below the title.
- **Hero Area**: Large illustrated hero area occupying the center using `hero_onboarding_search_shield.webp`.
- **Trust Indicators**: Three trust items placed in the lower third.
- **CTA**: Large blue Start Building button near the bottom.
- **Navigation**: No bottom navigation.

## Home Screen
- **App Bar**: Top app bar.
- **Header**: Map/exploration header occupying the top area using `header_home_map.webp`.
- **Title**: "What do you want to find?" overlaid or directly below the header.
- **Goal Cards**: 2-column goal-card grid.
- **Cards**: Six colorful goal cards with distinct accents.
- **Navigation**: Bottom navigation visible.

## Preview Screens
- **App Bar**: Top app bar.
- **Risk Header**: Hero header using `header_low_risk.webp` for Low Risk or `header_medium_risk.webp` for Medium Risk.
- **Search Panel**: Search string panel containing the raw output.
- **CTA**: Large copy button (Blue for Low Risk, Amber for Medium Risk).
- **Details**: Explanation/warning panel and protected chips grid.
- **Restriction**: No giant onboarding hero.
