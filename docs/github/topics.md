# Suggested GitHub Topics

Apply these topics to the repository via **GitHub → Settings → General → Topics**
(or the `gh repo edit` command below). They describe what the project is, not who
it is affiliated with — PokeQuery is an independent fan-made utility.

## Recommended topics

```
android
kotlin
jetpack-compose
pokemon-go
search-strings
offline-first
privacy-first
android-app
compose-ui
closed-testing
```

## Apply via CLI

```bash
gh repo edit chaglaruk/PokeQuery \
  --add-topic android \
  --add-topic kotlin \
  --add-topic jetpack-compose \
  --add-topic pokemon-go \
  --add-topic search-strings \
  --add-topic offline-first \
  --add-topic privacy-first \
  --add-topic android-app \
  --add-topic compose-ui \
  --add-topic closed-testing
```

## Suggested short description

```
Offline-first Android app that builds safer Pokémon GO search strings — no login, no tracking, no automation.
```

(Also saved in `docs/github/repo_description.txt`.)

## Notes

- Do **not** add topics that imply affiliation, e.g. `niantic`, `nintendo`,
  `official`, `pokemon` (use `pokemon-go` instead).
- Keep the topic list focused — 8–12 topics is ideal for discoverability.
