# Internal Testing Checklist

Before uploading the AAB to the internal testing track, ensure the following are complete:

- [ ] Version code incremented to `8`.
- [ ] Version name updated to `0.4.1`.
- [ ] AAB built as `app-release.aab` and verified.
- [ ] No debug information or dummy data remains in the UI.
- [ ] App uses exact `com.caglar.pokequery` package ID.
- [ ] No Pokémon IP in the app logo, banners, or screenshots.
- [ ] All features operate strictly offline.
