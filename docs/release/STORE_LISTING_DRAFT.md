# Store Listing Draft

**App Name:** PokeQuery: Safe Search Strings  
**Short Description:** Build safe, complex search strings for Pokémon GO offline.  
**Full Description:**  
PokeQuery helps you quickly organize and clean up your Pokémon GO storage. Create complex, safe search strings instantly without typing. Find exact IVs, prepare for 2x candy events, hunt for PVP candidates, and protect your rare favorites with one tap. 

PokeQuery operates entirely offline and requires zero permissions. It does not connect to your Pokémon GO account, ensuring total safety and compliance.

**Key Features:**
- Safe Cleanup and Candy Prep generators.
- Hundo, Nundo, and PVP (Great/Ultra League) filters.
- Supports both English and Turkish string output natively.
- 100% offline. Zero permissions. No ads.
