# PokeQuery v0.1.2 Real-World Validation Preparation

## 1. Purpose
Validate that generated PokeQuery search strings behave correctly inside the real Pokémon GO app.

## 2. Preconditions
* PokeQuery v0.1.1 UI polish installed
* Pokémon GO installed
* User has enough Pokémon variety to test shiny, legendary, costume, traded, favorite, tagged, untagged, 4★, low-IV, and duplicates where possible
* Never mass transfer based only on first result; review manually
