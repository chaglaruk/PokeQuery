# v0.2.0 Manual Validation Checklist

## Requirements
- Physical Android device with Pokémon GO installed.
- PokeQuery v0.2.0 installed.

## Validation Steps

### 1. Nundo Finder
- [ ] Open PokeQuery, select `Nundo Finder`.
- [ ] Verify generated string is EXACTLY: `0attack&0defense&0hp`.
- [ ] Verify Scope Meter shows `Very Narrow`.
- [ ] Copy and paste into Pokémon GO.
- [ ] **PASS CRITERIA**: Any Pokémon returned must appraise with exactly 0 Attack, 0 Defense, and 0 HP.

### 2. PvP Great League Candidate
- [ ] Open PokeQuery, select `PvP Candidates`, select `Great League`.
- [ ] Verify generated string is EXACTLY: `0-1attack&3-4defense&3-4hp&cp-1500`.
- [ ] Verify Scope Meter shows `Narrow`.
- [ ] Copy and paste into Pokémon GO.
- [ ] **PASS CRITERIA**: Returned Pokémon must be <= 1500 CP with low attack IVs and high defense/hp IVs.

### 3. PvP Ultra League Candidate
- [ ] Open PokeQuery, select `PvP Candidates`, select `Ultra League`.
- [ ] Verify generated string is EXACTLY: `0-1attack&3-4defense&3-4hp&cp-2500`.
- [ ] Copy and paste into Pokémon GO.
- [ ] **PASS CRITERIA**: Returned Pokémon must be <= 2500 CP with low attack IVs and high defense/hp IVs.

### 4. Lucky Trade Prep - Age Mode
- [ ] Open PokeQuery, select `Lucky Trade Prep`, select `Older Candidates`.
- [ ] Verify generated string is EXACTLY: `age365-&!traded`.
- [ ] Copy and paste into Pokémon GO.
- [ ] **PASS CRITERIA**: Returned Pokémon must have been caught more than 1 year ago and must not be marked as traded.

### 5. Lucky Trade Prep - Distance Mode
- [ ] Open PokeQuery, select `Lucky Trade Prep`, select `Distance Candidates`.
- [ ] Verify generated string is EXACTLY: `distance100-&!traded`.
- [ ] Copy and paste into Pokémon GO.
- [ ] **PASS CRITERIA**: Returned Pokémon must have been caught > 100km away (requires manual location verification) and must not be marked as traded.

### 6. Regression Testing
- [ ] Generate a standard `Safe Cleanup` string (without 0*).
- [ ] Verify string is `1*&!shiny&...`.
- [ ] Generate a `2x Candy Prep` string.
- [ ] Verify string is `count2-&!shiny&...`.
