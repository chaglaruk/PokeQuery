# v0.2.0 Power User Pack

## Overview
This update introduces powerful new search-string tools targeted at advanced Pokémon GO players (Power Users) without compromising the safety constraints established for casual users in earlier versions. 

## Key Additions
- **Scope Meter (Result Breadth)**: A new UI heuristic on the Preview screen that classifies strings as Very Narrow, Narrow, Moderate, Broad, or Very Broad. This helps set expectations before pasting a string into the game.
- **Nundo Finder**: A dedicated goal that safely targets exact 0/0/0 IV Pokémon (`0attack&0defense&0hp`).
- **PvP IV Candidates**: Support for Great League (< 1500 CP) and Ultra League (< 2500 CP) candidate filtering, focusing on low attack and high defense/HP IV spreads.
- **Lucky Trade Prep**: Filtering by age (e.g., > 365 days) or distance (e.g., > 100km) to identify prime candidates that might trigger a Lucky Trade or yield guaranteed XL Candy.

## Design Philosophy Maintained
PokeQuery remains a *guided assistant*, explicitly avoiding the path of becoming a complex sandbox UI (unlike full query constructors). Transfer protections remain strict for cleanup routines, while the new Power User features bypass those transfer protections to allow safe inspection (Info/Medium risk profiles).
