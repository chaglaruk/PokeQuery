# v0.3.0 UX Rebuild Plan

## 1. Current UX Problems
- **App Icon:** Missing a proper identity (defaults).
- **Onboarding:** Welcome screen is boring and lacks context.
- **Home Screen:** Wastes space, ugly generic title, uneven grid, repetitive 3-card structure.
- **Goal Flow:** Rigid 3-step process (Home -> Configure -> Review) forces unnecessary clicks.
- **GoalDetail Missing:** Options are generic toggles. Changes don't instantly reflect in the final string.
- **UI Elements:** Small touch targets for switches, primitive back animations.
- **Knowledge Base:** Primitive list, hard to read, no search.
- **Settings & Presets:** Empty settings, lack of real-world "power user" use-cases.

## 2. Target UX
"A guided Pokémon GO search-string assistant that instantly gives safe, useful, adjustable search strings."
- **Preset-First:** Common use-cases readily available.
- **Live Editable:** Settings update the string in real-time on the exact same screen.
- **One-Tap Copy:** Copy immediately without secondary validation unless risk is severe.

## 3. Screens to Rebuild
- **Home Screen:** Redesign with compact header, distinct Quick/Cleanup/Trade/Battle/Expert sections.
- **Onboarding Screen:** 3-step carousel/pager with hero visuals and explanation of privacy.
- **GoalDetail Screen (NEW):** Replaces `GuidedQuestionsScreen` and `PreviewScreen`. Live generated string, scope, risk, options.
- **KnowledgeBase Screen:** Add search bar, categorized collapsible cards, copy buttons.
- **Settings Screen:** Add meaningful settings (safety level, theme, duplicate defaults).

## 4. Risk Areas
- **State Management:** Tying live switches directly into the StringBuilderEngine reactively could introduce composition lag if not remembered correctly.
- **Navigation Animations:** The Navigation3 compose library is currently primitive. Implementing smooth transitions requires careful handling without breaking backstack.
- **Asset / App Icon creation:** Creating adaptive icons that don't violate IP.

## 5. Acceptance Criteria
- App purpose is obvious on first launch.
- App has a proper launcher adaptive icon.
- No forced "Review" button before string copy.
- Goal switches update strings live.
- Switch labels/rows are fully clickable.
- Cards are smaller, organized, with no ugly grid holes.
- Knowledge base is searchable and collapsible.
- Settings screen has meaningful options.
- Animations no longer feel primitive.
- Popular presets added and tested.
